# 项目说明

## 项目运行

```
npm install
```

```
npm run serve
```

## 项目打包

```
npm run build
```

## 同时也需要安装vant

```
npm install vant --save
```
    
## axios

```
npm install axios --save
```
    
## 由于压缩的是整体包下即可运行

```
npm run serve
```
